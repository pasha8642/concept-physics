# concept physics
The project performs addition, subtraction, division and multiplication of fractions.Converts from Celsius to Fahrenheit and vice versa.Converts gallons to liters and vice versa.Converts kilometers to miles and vice versa.
This project performs the action of a calculator, and it is useful to count fractions and what has already been mentioned.
First, to use the software, you need to download paycharm version 3.12 or higher, download the project and you can use it.
